# 📘 ΕΓΧΕΙΡΙΔΙΟ ΒΟΗΘΕΙΑΣ - ΠΡΟΣΟΜΟΙΩΤΗΣ ΔΙΚΤΥΩΝ
# ==============================================

## 🎯 ΒΑΣΙΚΕΣ ΛΕΙΤΟΥΡΓΙΕΣ

### 📍 ΠΡΟΣΘΗΚΗ ΣΥΣΚΕΥΩΝ
• [ΚΛΙΚ]: Κάντε κλικ σε συσκευή από το αριστερό πάνελ  
• [ΠΛΗΚΤΡΟΛΟΓΙΟ]: Ctrl + A (προσθήκη νέας συσκευής)  
• [ΜΕΤΑΚΙΝΗΣΗ]: Σύρετε συσκευές στο workspace  

### 🔗 ΣΥΝΔΕΣΗ ΣΥΣΚΕΥΩΝ
1. Πατήστε "Σύνδεση" 🔗
2. Επιλέξτε 1η συσκευή
3. Επιλέξτε 2η συσκευή
4. Οι συσκευές θα συνδεθούν αυτόματα

### ⚙️ ΡΥΘΜΙΣΕΙΣ ΣΥΣΚΕΥΩΝ
• [ΚΛΙΚ]: Σε συσκευή για πάνελ ρυθμίσεων  
• Περιλαμβάνει:
  - Διεύθυνση IP
  - Subnet Mask
  - Gateway
  - DNS Server
  - Domain Name (για servers)

## 🔧 ΕΡΓΑΛΕΙΑ ΔΙΚΤΥΟΥ

### 🚀 ΔΟΚΙΜΗ PING
1. Κάντε κλικ σε συσκευή
2. Πατήστε "Δοκιμή Ping" στο πάνελ
3. Επιλέξτε προορισμό
4. Το σύστημα θα προσομοιώσει ping
• **Εμφανίζεται αυτόματα η κονσόλα με το αποτέλεσμα.**
• Αν η επικοινωνία γίνεται μέσω router/gateway, στη διαδρομή θα εμφανίζεται ο router.

### 🛣️ ΔΟΚΙΜΗ ΔΙΑΔΡΟΜΗΣ
1. Πατήστε "Διαδρομή" 🛣️
2. Επιλέξτε 1η συσκευή
3. Επιλέξτε 2η συσκευή
4. Το σύστημα εμφανίζει τη διαδρομή (με όλα τα ενδιάμεσα hops, π.χ. router)

### 🌐 DNS RESOLUTION
• [Αυτόματο]: Κάθε συσκευή χρησιμοποιεί τον ρυθμισμένο DNS Server  
• [Χειροκίνητο]:  
  1. Πατήστε "Χειροκίνητο" 👆
  2. Επιλέξτε συσκευή πηγή
  3. Επιλέξτε DNS server
  4. Εισάγετε domain name προς αναζήτηση  
• **Η κονσόλα ανοίγει αυτόματα για επιτυχία ή αποτυχία DNS lookup (και στο manual DNS).**

### ▶️ ΠΡΟΣΟΜΟΙΩΣΗ ΚΥΚΛΟΦΟΡΙΑΣ
• Πατήστε "Προσομοίωση" ▶️  
• Δημιουργούνται τυχαία πακέτα  
• Οπτικοποίηση κυκλοφορίας
• Πατήστε ξανά για διακοπή ⏸️  

## 🌐 ΠΡΟΚΑΘΟΡΙΣΜΕΝΑ ΔΙΚΤΥΑ

### 🏠 LAN ΔΙΚΤΥΟ
• Πατήστε "LAN" για τοπικό δίκτυο:
  - Router (192.168.1.1)
  - Switch
  - Υπολογιστής (192.168.1.10)
  - Server (192.168.1.100)
  - DNS Server (192.168.1.53)

### ☁️ WAN ΔΙΚΤΥΟ
• Πατήστε "WAN" για εξωτερική σύνδεση:
  - Cloud/WAN (203.0.113.1)
  - Router με WAN/LAN
  - Πλήρες LAN δίκτυο
  - Static routing για WAN

## 💾 ΑΠΟΘΗΚΕΥΣΗ & ΦΟΡΤΩΣΗ

### 💾 ΑΠΟΘΗΚΕΥΣΗ WORKSPACE
1. Πατήστε "Αποθήκευση" 💾
2. Δώστε όνομα
3. Κατεβάζει JSON αρχείο

### 📂 ΦΟΡΤΩΣΗ WORKSPACE
1. Πατήστε "Φόρτωση" 📂
2. Επιλέξτε JSON αρχείο
3. Φορτώνει συσκευές και συνδέσεις

• [ΑΥΤΟΜΑΤΗ ΑΠΟΘΗΚΕΥΣΗ]: Κάθε 30 δευτερόλεπτα

## 🔍 ΕΝΤΟΛΕΣ ΔΙΑΧΕΙΡΙΣΗΣ

### 🌐 ΔΙΑΧΕΙΡΙΣΗ DNS
1. Επιλέξτε DNS server
2. Στο πάνελ ρυθμίσεων:
   • Προσθήκη νέων records
   • Διαγραφή records
   • Προβολή όλων των records

### 🚦 AUTO ROUTING
• Πατήστε "Routing" για αυτόματη ρύθμιση
• Όλες οι συσκευές χρησιμοποιούν το router ως gateway

### 🐛 DEBUG ΠΛΗΡΟΦΟΡΙΕΣ
• Πατήστε "Debug" για αναλυτικές πληροφορίες
• Εμφάνιση συνδέσεων στην κονσόλα
• Έλεγχος δικτύου για σφάλματα

## 🚀 ΟΔΗΓΙΕΣ ΓΡΗΓΟΡΗΣ ΕΝΑΡΞΗΣ

1. [ΓΡΗΓΟΡΗ ΕΝΑΡΞΗ]: Πατήστε το κουμπί για οδηγίες
2. [DEMO]: Χρησιμοποιήστε προκαθορισμένα LAN/WAN
3. [ΠΕΙΡΑΜΑΤΙΣΜΟΣ]: Προσθέστε & δοκιμάστε συνδέσεις
4. [ΕΚΜΑΘΗΣΗ]: Παρακολουθήστε πακέτα (Ping, Διαδρομή, DNS)

## ⚠️ ΣΥΜΒΟΥΛΕΣ ΑΝΤΙΜΕΤΩΠΙΣΗΣ ΠΡΟΒΛΗΜΑΤΩΝ

### 🔌 ΔΕΝ ΥΠΑΡΧΕΙ ΕΠΙΚΟΙΝΩΝΙΑ / ΠΡΟΒΛΗΜΑ ΣΤΙΣ ΣΥΝΔΕΣΕΙΣ
• Ελέγξτε IP διευθύνσεις, μάσκες, gateways  
• Βεβαιωθείτε για το ίδιο subnet όπου απαιτείται  
• Δείτε τη διαδρομή μέσω εργαλείου "Διαδρομή"

### 🌐 ΠΡΟΒΛΗΜΑΤΑ ΜΕ DNS
1. Βεβαιωθείτε ότι έχει οριστεί σωστός DNS server στη συσκευή
2. Ελέγξτε τη δ/νση IP του DNS server
3. Χρησιμοποιήστε το εργαλείο "Χειροκίνητο DNS" για δοκιμή

### ▶️ ΔΕΝ ΕΚΚΙΝΕΙ Η ΠΡΟΣΟΜΟΙΩΣΗ
• Βεβαιωθείτε ότι υπάρχουν ενεργές συνδέσεις
• Ελέγξτε αν οι συσκευές είναι online και έτοιμες
